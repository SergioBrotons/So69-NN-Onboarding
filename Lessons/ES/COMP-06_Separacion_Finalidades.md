## SECCIÓN 1 — PROPÓSITO ESTRATÉGICO

### Español (Primario)

Este módulo explica el principio de separación de finalidades (purpose limitation), que establece que los datos recopilados para un propósito específico no deben usarse para propósitos diferentes sin base legal apropiada. En el contexto de So69/AITANNA, esto significa que los datos capturados para análisis de tráfico no pueden usarse para otros propósitos sin consentimiento o base legal adicional.

### English (Controlled Translation)

This module explains the purpose limitation principle, which establishes that data collected for a specific purpose should not be used for different purposes without appropriate legal basis. In the So69/AITANNA context, this means data captured for traffic analysis cannot be used for other purposes without consent or additional legal basis.

## SECCIÓN 3 — EXPLICACIÓN NUCLEAR

### 3.1 Definición del Concepto

**Español:**

Separación de finalidades significa que cada uso de datos debe tener una base legal específica. Los datos recopilados para análisis de tráfico (bajo interés legítimo) no pueden automáticamente usarse para: marketing personalizado (requiere consentimiento), venta a terceros (requiere consentimiento), o identificación de individuos (requiere base legal diferente). Esta separación protege tanto a la organización como a los individuos.

**English Translation:**

Purpose limitation means that each use of data must have a specific legal basis. Data collected for traffic analysis (under legitimate interest) cannot automatically be used for: personalized marketing (requires consent), sale to third parties (requires consent), or identification of individuals (requires different legal basis). This separation protects both the organization and individuals.

### 3.2 Aplicación

| Propósito Original | Uso Propuesto | ¿Permitido? |
|-------------------|---------------|-------------|
| Análisis de tráfico | Métricas operativas | Sí |
| Análisis de tráfico | Marketing personalizado | No (requiere consentimiento) |
| Análisis de tráfico | Identificación | No (requiere base diferente) |
| Análisis de tráfico | Compartir con terceros | No (requiere consentimiento) |

## SECCIÓN 4 — ESCENARIO EJECUTIVO

### Situación de Presión

Un cliente propone: "Ya que tienen datos de tráfico, ¿podemos usarlos también para enviar ofertas a los visitantes?" Necesita entender la limitación.

### Respuesta Correcto

"No sin consentimiento adicional. Los datos fueron recopilados para análisis de tráfico bajo interés legítimo. Usarlos para enviar ofertas personalizadas es un propósito diferente que requiere consentimiento de los individuos. Nuestro sistema puede proporcionar análisis de tráfico; para marketing personalizado necesitaría un sistema diferente con el consentimiento apropiado."
