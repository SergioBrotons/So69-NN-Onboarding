## SECCIÓN 3 — EXPLICACIÓN NUCLEAR

### 3.1 Pasos de Instalación

| Fase | Actividades |
|------|-------------|
| Site survey | Evaluar ubicación, cobertura, interferencia |
| Instalación física | Montar sensores, verificar alimentación |
| Configuración | Ajustar parámetros, definir zonas |
| Calibración | Validar contra ground truth, ajustar |
| Go-live | Activar sistema, verificar operación |

### 3.2 Validación

Siempre validar con conteos manuales para confirmar precisión antes de entregar al cliente.

## SECCIÓN 4 — ESCENARIO EJECUTIVO

### Situación de Presión

El cliente quiere comenzar a usar el sistema inmediatamente después de la instalación.

### Respuesta Correcto

"Entiendo la urgencia, pero necesitamos completar la calibración primero. Sin calibración, los datos pueden no ser precisos. El proceso incluye: instalación, configuración, validación contra conteos manuales, y ajustes. Podemos tener el sistema listo en 24-48 horas."
